### 功能

使用BangumiApi获取视频信息并生成nfo文件  
批量重命名字幕文件  
ass字幕转srt（需要配置ffmpeg路径）  
添加RSS订阅和下载规则到qBittorrent（需要开启web-ui并禁用身份验证）  

### 测试环境：
Win11 .net8.0  JetBrains Rider 2024.3
